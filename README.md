# react-native-static-html-to-pdf

Convert static html file to pdf

## Installation

```sh
npm install react-native-static-html-to-pdf
```

## Usage

```js
import { multiply } from "react-native-static-html-to-pdf";

// ...

const result = await multiply(3, 7);
```
